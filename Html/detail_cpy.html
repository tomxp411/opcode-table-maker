65C02 Opcode Details: CPY

Compare Y to memory

SYNTAX MODE HEX LEN CYCLES FLAGS
CPY #$12 Immediate $C0 2 2 CZ----N
CPY $12 Zero Page $C4 2 3 CZ----N
CPY $1234 Absolute $CC 3 4 CZ----N

Compares the value in the Y register with the given value. It sets flags based on subtracting Y - _Value_.

- Sets C (Carry) flag if the value in Y is >= given value - Clears C (Carry) flag if the value in Y is < given value - Sets Z (Zero) flag if the values are equal - Clears Z (Zero) flag if the values are not equal - Sets N (Negative) flag if value in Y is < given value
