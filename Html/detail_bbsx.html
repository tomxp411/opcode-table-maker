65C02 Opcode Details: BBSx

Branch on Bit Set

SYNTAX MODE HEX LEN CYCLES FLAGS
BBS0 $12,$1234 ZP Relative $8F 3 5 -------
BBS1 $12,$1234 ZP Relative $9F 3 5 -------
BBS2 $12,$1234 ZP Relative $AF 3 5 -------
BBS3 $12,$1234 ZP Relative $BF 3 5 -------
BBS4 $12,$1234 ZP Relative $CF 3 5 -------
BBS5 $12,$1234 ZP Relative $DF 3 5 -------
BBS6 $12,$1234 ZP Relative $EF 3 5 -------
BBS7 $12,$1234 ZP Relative $FF 3 5 -------

Branch to LABEL if bit x of zero page address is 1 where x is the number of the specific bit (0-7).

Specific to the 65C02 (*unavailable on the 65C816*)

#### Example (ca65)

```asm @check_flag: BBS3 zeropage_flag, @flag_set @flag_not_set: NOP ... @flag_set: NOP ... ```

The above BBR3 looks at value in zeropage_flag (here it's a label to an actual zero page address) and if bit 3 of the value is *zero* the branch would be taken to `@flag_set`.
